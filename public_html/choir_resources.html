Note learning assistance: Sea Shanties

Please follow the links below to listen to a synthesized version of the piece. There is a general audio file (.ogg) with all four parts at equal volume and then four separate files with each of the four voice parts enhanced for ease of learning.

The files should play easily in either the Firefox or Chrome web browsers. However, if you are using Edge or Internet Explorer, you may not be able to access the files unless you also have VLC media player installed.

Edge and Internet Explorer users therefore have three options: download Chrome, download Firefox, or download VLC media player.

Safari users will also need to download VLC media player.
